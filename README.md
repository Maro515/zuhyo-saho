# 図表作法ドージョー

論文の図表を「触って、遊んで、腹落ちさせる」ためのインタラクティブ学習サイトです。
統計の基本から臨床試験・オミクス・機械学習まで、**全11章173項目**を収録しています。

🔗 **https://maro515.github.io/zuhyo-saho/**

## できること

各項目は次の4層で構成されています。

| | 内容 |
|---|---|
| 📘 **解説** | その手法が何のための道具か、考え方の要点 |
| 🏛 **歴史と原理** | 誰がいつ何のために生み出したかの年表、原理の数理・物理、中心となる数式、そして「原理を知ると図がどう読めるか」 |
| 🧪 **触って理解する** | ①操作ガイド → ②インタラクティブ図（SVG/JS で自作）→ ③観察ポイント・やってみよう課題 |
| ⚠️ **誤読しやすい点 / 🎯 クイズ** | よくある勘違いと、理解度の確認 |

- **⌘K で全文検索** — 見出しだけでなく本文・歴史・観察ポイントまで横断するので、「カプラン」「打ち切り」「過学習」のような語からでも該当項目に辿り着けます
- 左端の**目次レール**で現在位置を追跡、**← →キー**で前後の項目へ移動
- 進捗は localStorage に保存され、ホームの「続きから」で再開できます

## 技術構成

ビルドステップなしの静的サイトです。`index.html` を開くだけで動作します（`file://` でも可）。

```
index.html
css/style.css          デザインシステム（NEON AURORA / EDITORIAL）
js/app.js              ハッシュルーター・検索パレット・進捗管理・クイズ
js/charts.js           自作の軽量チャート描画ツールキット
js/widgets-chNN.js     項目ごとのインタラクティブ図（全173個）
data/chNN.js           各章の解説・注意点・クイズ
data/deep-chNN.js      各項目の「歴史と原理」
data/explore.js        各項目の「観察ポイント・やってみよう」
```

外部ライブラリ・CDN・ビルドツールへの依存はありません。
