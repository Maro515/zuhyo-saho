# 図表作法ドージョー

論文の図表を「触って、遊んで、腹落ちさせる」ためのインタラクティブ学習サイトです。
統計の基本から臨床試験・オミクス・機械学習まで、**全12章246項目**を収録しています。

🔗 **https://maro515.github.io/zuhyo-saho/**

## できること

各項目は次の4層で構成されています。

| | 内容 |
|---|---|
| 📘 **解説** | その手法が何のための道具か、考え方の要点 |
| 🏛 **歴史と原理** | 誰がいつ何のために生み出したかの年表、原理の数理・物理、中心となる数式、そして「原理を知ると図がどう読めるか」 |
| 🧪 **触って理解する** | ①操作ガイド → ②インタラクティブ図（SVG/JS で自作）→ ③観察ポイント・やってみよう課題 |
| ⚠️ **誤読しやすい点 / 🎯 クイズ** | よくある勘違いと、理解度の確認 |
| 📄 **代表論文** | その手法を実際に使った代表論文を各項目2本ずつ（全492本）。PubMed・DOIへ直接リンク |
| 🔧 **プロトコル・技術資料** | 手を動かすときの一次資料を各項目1〜3本（全654本）。プロトコル／メーカー資料／ソフトウェア／報告ガイドライン／データベースの5種別 |
| 🥋 **研究者の心得** | ページの締めくくり。**自分がその実験・解析をやる側**のときに心得ること（全246項目・631条） |

- **🔬 論文アナライザ** — 論文のPDF・本文テキスト・URLを読み込むと、使われている実験法・解析手法を自動抽出し、学ぶべき項目と学習ルートを提案します。**解析はすべてブラウザ内で完結し、論文は外部に送信されません**（PDF読み取りは同梱の pdf.js を使用）
- **⌘K で全文検索** — 見出しだけでなく本文・歴史・観察ポイントまで横断するので、「カプラン」「打ち切り」「過学習」のような語からでも該当項目に辿り着けます
- 左端の**目次レール**で現在位置を追跡、**← →キー**で前後の項目へ移動
- 進捗は localStorage に保存され、ホームの「続きから」で再開できます

## 技術構成

ビルドステップなしの静的サイトです。`index.html` を開くだけで動作します（`file://` でも可）。

```
index.html
css/style.css          デザインシステム（NEON AURORA / EDITORIAL）
js/app.js              ハッシュルーター・検索パレット・進捗管理・クイズ
js/charts.js           自作の軽量チャート描画ツールキット
js/widgets-chNN.js     項目ごとのインタラクティブ図（全173個）
data/chNN.js           各章の解説・注意点・クイズ
data/deep-chNN.js      各項目の「歴史と原理」
data/explore.js        各項目の「観察ポイント・やってみよう」
data/papers-chNN.js    各項目の代表論文（PubMed検証済み）
data/protocols-chNN.js 各項目のプロトコル・技術資料（全URL到達確認済み）
data/creed-chNN.js     各項目の「研究者の心得」
data/methods-syn-*.js  論文アナライザの手法検出用の同義語辞書
js/vendor/pdf.min.js   PDFテキスト抽出（pdf.js、同一オリジンに同梱）
```

外部ライブラリ・CDN・ビルドツールへの依存はありません。

## ライセンスと出典について

本サイトは『論文図表を読む作法 PREMIUM』（羊土社）の**項目立て・構成を参考にしつつ、解説文・図・インタラクティブ教材をすべてオリジナルに書き起こした個人の学習用教材**です。書籍の本文および図版の転載は一切していません。

第12章「発展編」は書籍に対応する章のない、本サイト独自の内容です。

掲載している代表論文（全492本）は、すべて PubMed API で実在・タイトル・発行年を機械的に照合したものです。プロトコル・技術資料のリンク（全654本）も、収録時に全URLへ実アクセスして到達を確認しています。ただし「その手法の代表例／参考資料として適切か」という選定の妥当性は筆者の判断によります。

外部サイトのURLは予告なく変更・削除されることがあります。リンク切れにお気づきの際はご容赦ください。

内容には誤りが含まれる可能性があります。研究・診療上の判断には、必ず原著論文や成書をご参照ください。
